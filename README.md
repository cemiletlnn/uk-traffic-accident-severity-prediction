# UK Traffic Accident Severity Prediction Using Machine Learning

## Project Overview

This project was developed as the final project for the Data Mining course at Eskişehir Osmangazi University.

The objective is to predict traffic accident severity using machine learning techniques and identify the most influential factors affecting accident severity based on the UK Traffic Accidents dataset.

---

## Dataset

- Source: UK STATS19 Traffic Accident Dataset
- Period: 2005–2014
- Number of records: 1,361,997

---

## Machine Learning Models

- Logistic Regression
- Random Forest
- XGBoost

---

## Methods

- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Label Encoding
- SMOTE
- Stratified 5-Fold Cross Validation
- SHAP Explainability
- K-Means Clustering
- Time Series Analysis

---

## Main Findings

- Best model: **XGBoost**
- ROC-AUC Score: **0.683**
- The most influential variables:
  - Number_of_Vehicles
  - Did_Police_Officer_Attend_Scene
  - Speed_limit
  - Road_Type
